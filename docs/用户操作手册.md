## 1. 系统概述

**团队协作平台**是一个基于 Web 的全栈项目管理系统，支持多用户同时在线协作。核心功能包括：

- **项目管理**：创建和管理多个项目，跟踪项目状态
- **任务看板**：可视化任务管理，拖拽切换状态
- **团队协作**：邀请成员、分配角色
- **文件共享**：上传、下载、预览项目文件
- **数据统计**：全局数据仪表盘

### 系统角色

| 角色 | 权限 |
|------|------|
| **管理员 (admin)** | 管理项目成员、修改项目信息、分配任务 |
| **成员 (member)** | 创建任务、上传文件、查看项目内容 |
| **观察者 (viewer)** | 仅查看项目内容 |

## 2. 快速开始

### 2.1 启动系统

**Windows 用户**：双击项目根目录下的 `start.bat`

**Mac/Linux 用户**：
```bash
chmod +x start.sh
./start.sh
```

启动成功后：
- 前端地址：`http://localhost:3000`
- 后端地址：`http://localhost:5000`

### 2.2 注册与登录

**首次使用需要注册账号**：

1. 打开 `http://localhost:3000`，点击"注册"
2. 填写用户名（至少3个字符）、邮箱、密码（至少6个字符）
3. 点击"注册"按钮
4. 注册成功后自动跳转登录页

**测试账号**（系统预置）：

| 用户名 | 密码 | 示例项目角色 | 说明 |
|--------|------|--------------|------|
| admin | admin123 | 项目管理员 | 可添加/移除成员、修改角色，推荐用于体验完整管理功能 |
| testuser1 | 123456 | 项目拥有者 | 项目创建者，拥有最高权限 |
| testuser2 | 123456 | 项目成员 | 普通成员，可参与任务和查看内容 |

> 系统首次启动会自动将三个测试账号加入示例项目，方便直接体验团队协作。

### 2.3 登录

1. 输入用户名或邮箱 + 密码
2. 点击"登录"
3. 登录成功后进入仪表盘首页

## 3. 功能模块详解

### 3.1 仪表盘（首页）

登录后默认进入仪表盘，显示：
- **快捷操作**：顶部快速入口，直达项目管理、任务看板、团队协作、文件管理、数据统计
- **统计卡片**：项目总数、任务总数、成员总数、文件总数、已完成任务数
- **最近项目**：按更新时间排序的项目列表
- **近期任务**：所有项目中的任务列表


### 3.2 项目管理

#### 查看项目列表
点击导航栏"项目"进入项目列表页。

#### 创建项目
1. 点击右上角"+ 新建项目"
2. 输入项目名称（必填）、描述（可选）
3. 选择初始状态：规划中 / 进行中 / 已完成 / 已归档
4. 点击"创建"

#### 编辑项目
鼠标悬停在项目卡片上，点击"编辑"按钮：
- 修改项目名称
- 修改项目描述
- 切换项目状态

#### 删除项目
鼠标悬停在项目卡片上，点击"删除"按钮，确认后删除。

**注意**：删除项目会同时删除该项目下的所有任务、成员关系和文件！

#### 项目详情页
点击项目卡片进入详情页，包含 4 个标签页：

**概览 Tab**：
- 项目基本信息
- 任务统计（待办/进行中/已完成）
- 成员数量
- 文件数量

**任务 Tab**：
- 查看该项目所有任务（看板视图）
- 创建新任务
- 拖拽或点击按钮切换任务状态
- 编辑/删除任务

**成员 Tab**：
- 查看项目成员列表
- 添加新成员（输入用户名）
- 修改成员角色
- 移除成员

**文件 Tab**：
- 上传文件
- 文件列表（名称、大小、上传者、时间）
- 点击文件名预览/下载
- 删除文件

### 3.3 任务看板

点击导航栏"任务"进入全局任务看板。

#### 看板布局
- **待办 (Todo)**：尚未开始的任务
- **进行中 (In Progress)**：正在执行的任务
- **已完成 (Done)**：已完成的任务

#### 创建任务
1. 选择一个项目
2. 点击任意列的"+ 新建任务"
3. 填写任务标题（必填）、描述（可选）
4. 选择优先级：低 / 中 / 高 / 紧急
5. 指派负责人（可选）
6. 点击"创建"

#### 移动任务
- **方法一**：拖拽任务卡片到目标列
- **方法二**：点击卡片上的状态按钮

#### 编辑任务
点击任务卡片的标题或描述区域进行修改。

#### 删除任务
点击任务卡片上的"删除"按钮。

### 3.4 团队协作

点击导航栏"团队"进入团队管理页。

#### 添加成员
1. 选择一个项目
2. 在"添加成员"区域输入用户名
3. 选择角色：管理员 / 成员 / 观察者
4. 点击"添加"

#### 修改角色
在成员列表中，点击角色下拉框切换角色。

**注意**：只能修改角色低于自己的成员。

#### 移除成员
点击成员右侧的"移除"按钮。

### 3.5 文件管理

点击导航栏"文件"进入文件管理页。

#### 上传文件
1. 选择一个项目
2. 点击"上传文件"按钮
3. 选择本地文件
4. 等待上传完成

**限制**：单个文件最大 50MB

#### 下载/预览文件
点击文件名，在新标签页中打开（浏览器支持的文件格式可直接预览）。

#### 删除文件
点击文件右侧的"删除"按钮。

### 3.6 数据统计

点击导航栏"统计"进入数据统计页。

- 全局概览：项目数、任务数、平台用户数（注册用户数）、文件数、团队成员数
- 任务状态分布：待办/进行中/已完成
- 项目状态分布：规划中/进行中/已完成
- 任务优先级分布：低/中/高/紧急

## 4. 常见问题

### Q: 忘记密码怎么办？
A: 当前版本暂不支持自助找回密码，请联系管理员在数据库中重置。

### Q: 文件上传失败？
A: 检查文件大小是否超过 50MB，检查后端服务是否正常运行。

### Q: 页面显示异常？
A: 刷新页面（F5），如仍有问题请清除浏览器缓存后重试。

### Q: Token 过期？
A: 系统会自动使用 Refresh Token 刷新，如仍然失败请重新登录。

### Q: 如何创建多个项目？
A: 在项目列表页点击"+ 新建项目"，可以创建任意数量的项目。
