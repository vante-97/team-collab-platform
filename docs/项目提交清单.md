## 六、老师拿到项目后的操作

### 1. 安装运行

```bash
# Windows: 双击 start.bat
# Mac/Linux:
chmod +x start.sh
./start.sh
```

### 2. 访问系统

- 前端：`http://localhost:3000`
- 后端：`http://localhost:5000`

### 3. 测试账号

| 用户名 | 密码 | 平台角色 | 示例项目角色 |
|--------|------|----------|--------------|
| admin | admin123 | 平台管理员 | 项目管理员（可添加/移除成员、修改角色） |
| testuser1 | 123456 | 普通用户 | 项目拥有者 |
| testuser2 | 123456 | 普通用户 | 项目成员 |

> 用 `admin` 登录可直接体验团队协作的完整管理功能（添加成员、移除成员、修改角色）。

### 4. 查看文档

- 项目说明：`README.md`

- 数据库设计：`docs/数据库设计文档.md`

## 七、技术亮点总结（供老师参考）

1. **前后端分离架构**：Next.js 14 App Router + Flask REST API
2. **TypeScript 全栈类型安全**：前端完整类型定义，零 TS 错误
3. **JWT 双 Token 认证**：Access Token + Refresh Token 自动续期
4. **暗色玻璃态设计系统**：统一 CSS 变量 + Tailwind 自定义主题
5. **完整的 CRUD 操作**：项目、任务、成员、文件全覆盖
6. **看板式任务管理**：三列状态 + 拖拽移动
7. **角色权限控制**：管理员/成员/观察者三级权限
8. **文件上传下载**：multipart 上传 + 文件预览
9. **速率限制**：登录/注册 API 防暴力破解
10. **一键启动脚本**：Windows + Linux/macOS 双平台支持
11. **完整的项目文档**：操作手册 + 数据库设计 + API 文档 + 截图
12. **移动端响应式**：导航栏汉堡菜单 + 卡片自适应布局
